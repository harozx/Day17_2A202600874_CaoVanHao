Giả Thuyết Cốt Lõi

Hypothesis
"Khách hàng mua/thuê nhà muốn được tự tra cứu thông tin quỹ căn hộ và tự tính toán dòng tiền trả góp vay ngân hàng một cách riêng tư (qua chatbot AI) trước khi họ sẵn sàng chia sẻ thông tin liên hệ cho nhân viên Sales."
  • Giải quyết nỗi sợ của khách hàng: E ngại việc bị lộ số điện thoại dẫn đến bị telesale spam liên tục khi mới chỉ đang tìm hiểu sơ bộ.
  • Mở rộng đầu phễu khách hàng: Cung cấp một môi trường tương tác an toàn giúp gia tăng tỷ lệ sẵn sàng cung cấp thông tin (opt-in rate) khi họ thực sự có nhu cầu đặt lịch xem nhà.

Cách Nhóm Đang Triển Khai

Current Execution

Hệ thống hoàn chỉnh nhóm đang xây dựng gồm 3 phân hệ độc lập:

  • Giao diện: React Zalo Mini App chạy mượt mà trên di động, tích hợp Zalo SDK xác thực người dùng.
  • AI Engine: FastAPI Python + Gemini AI sử dụng cơ chế Function Calling để tính toán lãi suất và truy vấn thông tin căn hộ chính xác.
  • Backend: Java Spring Boot + database MySQL online trên cloud Railway.
⚠️ Hạn chế để test nhanh: Tốn thời gian phát triển (3-4 tuần), tốn phí vận hành server Cloud (Railway, Database, Zalo verification), bảo trì hệ thống phức tạp.
